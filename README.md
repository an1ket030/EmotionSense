
# EmotionSense: Real-Time Human Detection & Emotion Recognition System

This project uses deep learning and computer vision to detect human faces and classify their emotional expressions in real-time.

## Features
- Real-time face detection (OpenCV)
- Emotion classification (CNN trained on FER-2013 dataset)
- Streamlit UI for live demo

## Run the App
```
pip install -r requirements.txt
streamlit run app/streamlit_app.py
```
